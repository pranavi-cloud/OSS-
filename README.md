## Project Overview
This repository contains a set of Bash scripts developed for an Open Source Software (OSS) audit. The target software for this audit is **Git**. The scripts automate system identification, dependency checks, and log analysis.

## File Descriptions
| File | Purpose |
| :--- | :--- |
| `script1.sh` | System Identity & Uptime Report |
| `script2.sh` | Software Installation & Version Check |
| `script3.sh` | Log Integrity & Error Scanning |
| `script4.sh` | File Permissions & Security Audit |
| `script5.sh` | Final Project Declaration |

## How to Execute
To run the audit scripts, use the following commands in a Linux terminal:
1. Make the scripts executable: `chmod +x *.sh`
2. Run the main identity script: `./script1.sh`
